# Flor de Sal - Site

Site estático ajustado ao briefing de desenvolvimento do projeto Flor de Sal.

## Estrutura

- `index.html`: página inicial.
- `pages/`: páginas internas do site.
- `css/`: estilos globais.
- `js/`: scripts de navegação, filtros e reservas.
- `img/`: imagens oficiais, incluindo `logo.png`.
- `assets/illustrations/`: ilustrações auxiliares do hero e cardápio.
- `docs/`: briefing e plano de desenvolvimento.

## Páginas

- `pages/cardapio.html`: cardápio com categorias, imagens locais, preços e filtros.
- `pages/about.html`: história, chef/equipe e filosofia da cozinha.
- `pages/reservas.html`: formulário de reserva com validação de data, horário e pessoas.
- `pages/contato.html`: contato, horários, redes sociais e mapa.

## Identidade aplicada

- Logo oficial: `img/logo.png`.
- Nome preservado: Flor de Sal - Gastronomia Contemporânea.
- Paleta do briefing: `#2C2C2A`, `#854F0B`, `#EF9F27`, `#FAEEDA`, `#F1EFE8`.
- Títulos em Georgia e corpo em Inter/Calibri.

## Recursos implementados

- Fotos reais externas como placeholders profissionais para pratos e ambiente.
- Reserva via WhatsApp com mensagem preenchida automaticamente.
- Cardápio com filtros por categoria, preço e tags.
- Home com experiência da casa, depoimentos e chamada para Instagram.
- SEO local com dados estruturados de restaurante.
- Animações leves de entrada, hover e microinterações.

## Como testar

Abra `index.html` no navegador. Como o projeto é estático, não é necessário iniciar servidor local.
